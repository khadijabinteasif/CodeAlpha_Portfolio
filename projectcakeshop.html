Cake Shop Website

Front-End Technologies:

  • HTML, CSS, and Bootstrap: Used to create a responsive design that looks great on all devices.
  • JavaScript: Added interactivity to enhance the user experience.

Back-End Technologies:
  • PHP: Handles server-side logic to manage user requests and process orders.
  • MySQL: Manages data securely, including user information and transactions.

Features:
  • Reviews and Brand Details: Customers can read reviews and learn more about the brand.
  • About Us & Contact: Provides a comprehensive About Us page, contact information, and an integrated map for easy navigation.
  • Billing Cart: Calculates total bills to ensure a smooth and accurate checkout process.
  • User Authentication: Includes login and sign-up functionality for customers to track their orders.
